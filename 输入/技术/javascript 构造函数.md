# javascript 构造函数

## 1 构造函数和普通函数的区别

### 1.1 定义区别

**构造函数和普通函数的区别不是在函数定义时决定的，而是在函数调用时决定的，一个函数，只有当它是被 `new` 操作符调用时它才是构造函数，否则它就是普通函数，这和它的函数名是否首字母大小写没有任何关系；**
